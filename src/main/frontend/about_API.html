Endpoints

Retrieve array of symbols and corresponding market caps (getSymbols)
Returns an array of stock symbols and corresponding market caps for a pre defined list of popular companies in the requested sector

Path parameters

sector (required)
Path Parameter — Sector that company is established and active in. Currently accepted sectors: 'tech', 'medical', 'finance' and 'energy'.

Return type

List of symbols

Example data

Content-Type: application/json

[ 
    {
        "symbol": "AAPL",
        "marketcap": 2869807000000
    }, 
    {
        "symbol": "GOOG",
        "marketcap": 1827388000000
    } 
]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

List of symbols with market cap

400

Sector doesn't exist

429

Too many backend API-requests, try again soon

Retrieve stock and tweets (getStockTweet)
Returns stock information combined with an array of associated tweets

Path parameters

stock_symbol (required)
Path Parameter — Symbol or name of a company listed on the US stock market

Return type

StockTweet

Example data

Content-Type: application/json

{
    "stock": {
        "symbol": "AAPL",
        "percentChange": 0.0988,
        "companyName": "APPLE INC",
        "priceUSD": 172.17,
        "priceSEK": 1562.8
    },
    "tweets": {
        "data": [
            {
                "authorName": "krizdabz",
                "id": "1479782908386754565",
                "text": "\"Nearly 1.65B people across the globe connect to the internet through an Apple product, like the iPhone, iPad or iMac.\" Go, #AAPL!",
                "tweetId": "1479782908386754565"
            },
            {
                "authorName": "financemaster",
                "id": "1479755395279331329",
                "text": "RT @financemaster: Building a dividend portfolio that pays you monthly couldn’t be simpler;\n\n- #AAPL\n- #NKE\n- #MCD\n\nThese 3 stocks combined…",
                "tweetId": "1479755395279331329"
            }
        ]
    }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Stock and list of tweets

429
